\doxysubsection{Detailed Description}
\mbox{\hyperlink{classafarcloud_1_1_planner}{Planner}}\+: A class that implements the base class for every planner algorithm. This is an abstract class because of protected constructors. Each derived class must implement its own constructor. Moreover, each derived class must implement the planning function.
